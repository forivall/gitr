gitr: recursive git command line tool

  • https://npmjs.org/package/gitr

Install:

sudo npm install -g gitr

Example usage:

gitr status
gitr pull
gitr "log --since '1 day ago' --oneline --pretty=format:'%s'"

Building:

git clone git://github.com/kenglxn/gitr.git
cd gitr
npm test
